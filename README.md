# Load Balancer ⚖️

This repository contains basic load balancer for HTTP applications

## Contains
- ORM support
- MySQL support
- Configuration System

## Setup

1) If you want to use this load balancer, you just need to compile or run this micro-service.

2) Then setup mysql configuration.. When you will run micro-service should migrate ``Service`` struct to SQL (ORM - Object-Relational Mapping)
